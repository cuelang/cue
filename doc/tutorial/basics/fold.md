# Folding of Single-Field Structs

CUE allows a shorthand for structs with single members.

<!-- CUE editor -->
_fold.cue:_
```
outer middle inner: 3
```

<!-- JSON result -->
`$ cue export fold.cue`
```json
{
    "outer": {
        "middle": {
            "inner": 3
        }
    }
}
```
