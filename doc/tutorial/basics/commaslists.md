# Commas are Still Required in Lists


Commas are still required as separators in lists.
The last element of a list may also have a comma.

<!-- CUE editor -->
```
[
    1,
    2,
    3,
]
```

<!-- JSON result -->
```json
[
    1,
    2,
    3
]
```
